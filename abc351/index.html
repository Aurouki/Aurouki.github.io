Atcoder Beginner Contest 351

A - The bottom of the ninth(模拟)

题意

    两支队伍A,B正在进行比赛,比赛分为上半和下半两场各有9局,已知上半局A的全部得分和下半局B的前8局得分,问B的最后一局得分至少是多少才能使得B的总分严格大于A?

思路

    设所有A的总和sum_A= $\sum_ {i=1}^9 A_{i}\quad$,前八局B的总和sum_B为$\sum_ {i=1}^8 B_{i}$;易知，当sum_A$\geq$sum_B时,此时B需要sum_A-sum_B+1的得分，反之，B已经严格高于A的得分.

B - Spot the Difference(模拟)

题意

两个N$\times$N的网格，已知这两个网格有且只有一个位置(i,j)内的字符不同。请找出这个(i,j)的位置，行列的下标从1开始编号。

数据范围

  • $1 \leq N \leq 100$
  • $A_{i,j}$,$B_{i,j}$均为小写字母

思路

由于N的范围极小，输入两个方格后直接按行遍历即可，找到第一个不同的字符位置就是答案.

C - Merge the balls(栈)

题意

    拥有N个球，其中每个球的体积大小为$2^{A_{i}}$,从i=1开始依次将球装进盒子里。每次装完球后执行以下操作：

  1. 若当前只有一个球，则结束操作
  2. 若最靠右的两个球体积不等，则结束操作
  3. 若最靠右的两个球体积相等，则将其合并为一个球，体积翻倍，并返回步骤1重新操作；

数据范围

  • $1$ $\leq$ $N$ $\leq$ $2\times10^5$
  • $0$ $\leq$ $A_{i}$ $\leq$ $10^9$

思路

    由于每次是从右往左依次合并，符合“后进先出”的特征，所以可以用栈去模拟合并的过程。对于每次的合并，显然只有相同的指数可以进行相加，又$2^i$+$2^i$=$2^{i+1}$，故主需要在栈内丢入指数，合并时将指数加一。最终输出栈内的元素数量即可.
    虽然N高达$2\times10^5$，但是从感性方面去理解，每合并一次数量减一，从最坏情况去考虑，加入一个球后前面全部进行合并，但从该球往后整个盒子内的球数再度变为1…所以整体的复杂度还是O(N)量级的

F - Double Sum(树状数组+离散化)

题意

    给定一个整数序列 A=($A_1$,$A_2$…$A_N$),计算以下表达式：
$\sum\limits_{i=1}^{N}\sum\limits_{j=i+1}^{N}max(A_j-A_i,0)$

数据范围

  • $2 \leq N \leq 4\times10^5$
  • $0 \leq A_i \leq 10^8$
  • 保证答案小于$2^{63}$

思路

    形式化地，$\sum\limits_{i=1}^{N}$$\sum\limits_{j=i+1}^{N}$$max(A_j-A_i,0)$ 等价于当i < j 时，所有$A_j>A_i$的逆序对的差值和，其中$i\in[1,n-1],j\in[i+1,n]$.顾名思义，设对于任意的i，其逆序对的数量为num,则其对答案的贡献为$\sum\limits_{j=i+1}^{N}A_j[A_j>A_i]-numA_i$.所以我们需要知道其数量和对应的*值的总和.
    从逆序对数量的角度出发，我们以往可以通过树状数组模拟桶去维护小于等于某个数的数量，同样，我们也可以去维护小于某个数的总和；对于逆序对的求解，我们选择从i-1开始倒序模拟本题需要求的条件是大于，所以需要进行转换，大于$A_i$的数量只需要用区间的总数量减去小于等于$A_i$的总数量，即$(N-i)-\sum\limits_{j=i+1}^{N}1[A_j \leq A_i]$,同理，大于$A_i$的总和为$\sum\limits_{j=i+1}^{N}A_j-\sum\limits_{j=i+1}^{N}A_j[A_j \leq A_i]$，其中，区间的总和可以用前缀和实现.
    同时，因为$A_i$的值最高的值为$10^8$，已经远远超过了桶的承受范围，所以需要对$A_i$进行离散化操作，但是求和的时候必须用到的是真实值，所以还需要再开一个数组保存原始数值，以便在求和的时候通过下标索引到真实值.
